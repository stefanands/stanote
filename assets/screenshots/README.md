# Captures d'écran

Déposez ici les images affichées dans le README.

- **`apercu.png`** — capture principale (large, ~1600 px de large idéalement),
  affichée dans la section « Aperçu ».

Optionnel (décommenter le bloc correspondant dans le README racine) :

- `editeur.png` — l'éditeur markdown
- `terminal.png` — le terminal intégré
- `dispositions.png` — les différentes dispositions

Astuce : sur macOS, `⇧⌘4` puis Espace capture une fenêtre entière avec son ombre.
Une fois les images ajoutées ici, elles s'afficheront automatiquement sur GitHub.
